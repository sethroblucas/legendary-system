# AETHON

Real-time planetary intelligence visualization system. Models and monitors orbital assets and unregistered fragments through a sculpted holographic projection interface.

Built with React, React Three Fiber, and SGP4 orbital propagation against live CelesTrak TLE data.

## Stack

- React + TypeScript + Vite
- React Three Fiber / Drei / Postprocessing
- satellite.js (SGP4)
- Zustand
- Framer Motion

## Run

```
npm install
npm run dev
```
